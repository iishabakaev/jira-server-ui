# @app/web

React 19 SPA. Bundler — Vite 6. Routing — TanStack Router (file-based,
generates `routeTree.gen.ts`). Styling — Tailwind v4 via native plugin.

Transport: a single Eden Treaty client at `src/lib/eden.ts`. No `fetch`
or `axios` in components.

See `docs/specs/02-project-structure.md` for the feature folder rules.
